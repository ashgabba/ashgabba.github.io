Musical Tone Generation
System in Verilog HDL

Generates a sine wave look-up-table that is multiplied with an ADSR (attack-decay-sustain-release) waveform to produce a characteristic musical tone. This project was created with modular design approach that makes the system easily scalable for more complexity and can be modeled with a simple block diagram for each module.
